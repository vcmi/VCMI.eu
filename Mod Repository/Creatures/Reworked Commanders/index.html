    Reworked Commanders

    Description

    Reworked commanders for original factions. Loosely based on abilities and design of WoG commanders.

    Version

    1.28

    Author

    Concept and most Commanders' graphics - WoG Team; New Commanders Mod - Avatar support and MinosKing; Werewolf def - MDT; Orange Orc def - HotA Team; Idea and implemetation - Andruids; Sunseeker def - Andruids, based on Kuririn's Crusader; Totem def - Andruids, based on Rettosukero's Wendigo.
