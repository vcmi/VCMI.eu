    New Skills Pack

    Description

    Adds 5 new secondary skills:
     Anatomy chance to deal double damage
     Fury increases number of enemies slain
     Impact reduces enemy defense
     Magic Fader blocks high-level spellcasting
     Nimbleness lowers enemy attack through dodging

    Each skill features unique animations, effects, and specialized heroes.

    Version

    1.0.7

    Author

    FallainBlack (Fury, Anatomy, Impact, Nimbleness), avatar (Magic fader)

    Contact

    https://discord.gg/PDXDf8P8
