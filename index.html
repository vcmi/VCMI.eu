    VCMI

    VCMI is an open-source engine for Heroes III, offering new and extended possibilities. To use VCMI, you need to own the original data files.

    Download News Forum
    Go to VCMI-Discord

    GUI improvements

    high resolutions, stack queue, creature window...

    Mod support

    add new towns, creatures, heroes, artifacts and spells without limits or conflicts

    Launcher

    easy configuration - download mods from our server and install them immediatelly

    Multi Platform

    Windows, Linux, macOS, iOS and Android
